# Genre Analysis

This directory contains the code and analysis for the release strategies.


## Files in analysis_codes

*   analysis.py : Python scripts for data processing and release strategy (streaming vs theatretical) analysis.
*   dkw.py: Python scripts for dkw-band plot generation for runtime.
*   genre_plot.py: Python scripts for genre visualizations for release strategies.
*   paper_monthly_plots: Python scripts for time-series visualizations that were used in paper.

previous_unused_codes folder contains analysis codes that were discarded for various reasons (old dataset - change of experiments etc.) - includes heavy usage of LLM's and CO-pilot

## How to Run

It is recommended to run scripts using `uv` in order to ensure proper workflow.
